# Sail Simulator (Spek Kit Experiment)

This project is a Sail Simulator developed as part of a Spec Kit experiment. The goal is to explore and demonstrate the use of Spek Kit for collaborative specification-driven development.

## Project Structure

- **app/**: Contains the main application source code, configuration, and documentation.
- **specs/**: Contains specifications, plans, and tasks for the project, written in markdown.

## Getting Started

For setup instructions, usage, and development guidelines, see the [App README](app/README.md).
